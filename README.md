# duotetris
tetris for two

## Controls
space: start and restart game

### Player 1
left, right: move left, right
down: soft drop
up: hard drop
<, >: spin counterclockwise, clockwise
rshift: hold piece

### Player 2
a, d: move left, right
s: soft drop
w: hard drop
c, v: spin counterclockwise, clockwise
lshift: hold piece
